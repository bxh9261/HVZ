# Hockey vs Zombies
Asteroids is a game I made in Unity, a simulation showcasing collision detection and autonomous agents.

## Game controls
* SPACE – shoot
* UP – accelerate
* LEFT and RIGHT – rotate

## Features
* 3D Models
* Game Mode and Simulation Mode
